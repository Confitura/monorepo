TRACK A

Blockchain coding in Java

Today blockchain technology and cryptocurrencies are very hot topic. It is good to know how this database works and in which systems it may or may not be useful. During the workshop we will launch a private blockchain instance using the Ethereum implementation and prepare a client application in Java that will use it for data storage. We will also dispel the myths around blockchain.

Introduction to streaming data and stream processing with Apache Kafka

Apache Kafka has become the leading data streaming technology and is used in production by over 2000 companies like Netflix, Airbnb, Uber and LinkedIn. Kafka is used for building real-time data pipelines and streaming apps. It is horizontally scalable, fault-tolerant and very fast. This workshop will explore the basic concepts of messaging systems based on publish - subscribe messaging in Kafka.

TRACK B

Apache Spark lesson 2

So you've went through all the word counts and other hello-world type of codes, but you have some real data to analyse and you found answers to your issues missing we can help! This tutorial goes beyond the basic examples and shows how to deal with common issues of Data Scientist and Engineers using Spark for real (big) data sets. We focus on the Spark SQL, i.e. DataSets and DataFrames, show why it's good for both Java/Scala nad Python users, how to avoid typical errors and how to deal with them.

Introduction to Natural Language Processing

Natural Language Processing (NLP) is gaining a lot of popularity in recent years, thanks to advancements in state-of-the-art methods and algorithms, processing power available and rising expectations of end users of technology products. During this workshop we will make the first steps towards processing natural language, discuss the main stages necessary to perform typical tasks and make an overview of available methods and tools for processing English and Polish. Finally, we will talk about current research directions and perspectives for months and years to come.
